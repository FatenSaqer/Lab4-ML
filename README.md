# Lab4-ML
This lab focused on improving data quality and preparing the dataset for machine learning by applying several preprocessing techniques. First, the dataset was examined to ensure correct data types and identify potential issues such as missing values and outliers. Since the dataset did not originally contain missing values, artificial ones were introduced and handled using median imputation due to its robustness against outliers. Outliers were then detected using the Interquartile Range (IQR) method and removed to reduce their impact on analysis. Next, numerical features were normalized using both Min-Max scaling and Z-score standardization to ensure consistent feature scales. A correlation heatmap showed that most features have weak relationships, indicating limited redundancy, and therefore PCA was applied mainly for demonstration purposes rather than necessity. Overall, the dataset was successfully cleaned, standardized, and prepared for further machine learning tasks.
